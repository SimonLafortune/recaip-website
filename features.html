How the AI reads your code

Recaip does not rely on commit messages alone. It reads multiple data sources to produce accurate, context-aware output.

Reads the diff

Recaip parses the actual code changes in your PR. It identifies what files changed, what logic was added or removed, and whether the change is user-facing or internal.

Pulls context from tickets

If your PR links to a Linear, Jira, or GitHub issue, Recaip reads the ticket title and description. This adds business context the code alone does not provide.

Learns your voice

Every time you edit a draft before approving it, Recaip records the difference. Over time, it adjusts its writing style to match yours. The more you edit, the fewer edits you need.
